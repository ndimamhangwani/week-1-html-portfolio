## Ndima Karabo Mhangwani Portfolio
This is my personal portfolio website built using HTML5 and semantic elements.  
It showcases who I am, my skills, and the projects I am working on as I grow as a developer.


## Sections
- **Header & Navigation** – Site title and navigation links  
- **About Me** – Short introduction and background  
- **Projects** – A list of projects I have worked on  
- **Skills** – Technologies and tools I use  
- **Gallery** – Images representing my work  
- **Contact** – Email and GitHub links  

## Technologies Used
- HTML5
- Semantic HTML
- Git & GitHub

## Features
- Clean and simple layout  
- Semantic HTML elements  
- Accessible images with alt attributes  
- Easy to read and maintain  

## How to Use
1. Clone the repository
2. Open the `index.html` file in your browser
3. Explore the portfolio sections
